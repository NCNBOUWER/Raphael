# Raphael Physics

This folder contains the classic/current Raphael simulation core for natural phenomena and simulations. It intentionally does not import N^3 or EML#F.

## Run

```bash
python3 Raphael_Runner.py --validate
python3 tests/test_classic_raphael.py
```

## Included modules

- `natural_phenomena/fractal_expansion.py`
- `natural_phenomena/time_dynamics.py`
- `natural_phenomena/system_interaction.py`
- `Simulations/data_io.py`
- `Simulations/visualization_tools.py`
- `Simulations/simulation_controller.py`
- `raphael_equations.py`
- `Raphael_Runner.py`

## Boundary

This is a clean classic Raphael package. It is not N^3, not EML#F, and not Raphael Lite.
