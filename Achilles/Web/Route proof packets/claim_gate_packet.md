# Claim gate packet

Purpose: review route-facing claims before preview, paste, or public exposure.

## Required fields

| Field | Value |
|---|---|
| Claim text | TBD |
| Source file | TBD |
| Evidence class | TBD |
| Review state | pending |
| Allowed wording | TBD |
| Blocked wording | TBD |
| Public-safe decision | pending |

## Checks

- Claim text is copied exactly from proposed route copy.
- Source file is named and traceable.
- Evidence class is stated before public wording is approved.
- Unsupported validation or empirical claims remain blocked.
- Allowed wording is weaker than or equal to the available evidence.
- Public-safe decision is recorded before route release.

## Routing

Use this packet for any route copy involving proof status, validation state, empirical strength, investor-facing claims, scientific claims, or system capability claims.
