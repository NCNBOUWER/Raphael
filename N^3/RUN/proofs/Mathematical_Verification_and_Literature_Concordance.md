# Mathematical verification and literature concordance

## Internal verification

The package verifies the finite probability core through direct enumeration, the 2d6 count function, product probability, exact rational normalization, unique mode detection, minimum corner-state detection, barycentric closure checks, and Gamma interpolation closure checks.

## Literature posture

Named references are not used as proof substitutes. Background references may support finite probability, simplex geometry, entropy, group/orbit structure, and dimensional analysis. Exact page and section sign-off remains an open task in the live task register.

## Citation handling

Incomplete citation material is not treated as a completed front-facing citation register. It is tracked in `INTERNAL/registers/OPEN_TASKS.xlsx` until completed.

## Proof classes

| Class | Meaning |
|---|---|
| P0 | direct finite enumeration |
| P1 | algebraic derivation from enumeration |
| P2 | computational verification |
| P3 | geometric or compositional visualization |
| P4 | analogy or correspondence |
| P5 | hypothesis or audit-pending |
| Blocked | not allowed as a publish claim |
